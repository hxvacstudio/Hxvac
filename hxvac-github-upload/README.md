# Hxvac Portfolio Website

Static website files for the Hxvac Roblox trailer and Shorts portfolio.

## Upload Notes

Upload these files to the root of the `hxvacstudio/Hxvac` GitHub repository:

- `index.html`
- `styles.css`
- `script.js`
- `Logo.png`

The original trailer and Short videos are not included in this upload package because each file is over GitHub's normal 100 MB file limit. Once the videos are hosted somewhere else, replace the video URLs in `index.html`:

- `./media/hxvac-trailer.mov`
- `./media/hxvac-short.mov`

Recommended video hosting options: YouTube, Google Drive direct embed, Discord CDN, or a compressed MP4 under 100 MB.
